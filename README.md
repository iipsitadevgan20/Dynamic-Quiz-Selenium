# Dynamic Quiz Application with Selenium Automation

## Objective
A dynamic quiz application built using HTML, CSS, and JavaScript, automated and tested using Selenium WebDriver (Python).

## Features
- Difficulty selection (Easy, Medium, Hard)
- Countdown timer per question
- Dynamic question loading
- Result analysis with charts
- Responsive UI (Desktop, Tablet, Mobile)

## Testing
- Automated using Selenium WebDriver
- Random difficulty selection
- Random answer selection
- Execution logs stored in `selenium_test.log`

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js
- Python
- Selenium WebDriver

## How to Run
1. Open `quiz.html` in Chrome
2. Or run automation:
   ```bash
   python quiz_test.py
